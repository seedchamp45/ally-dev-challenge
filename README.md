# Ally Dev Challenge

### Tech Stack
- [Expo](https://expo.dev/) 
- [ReactNative](https://reactnative.dev/) 
- [TypeScript](https://typescriptlang.org)

## Getting Started

Install dependencies
````bash
npm install
````

Launch the development apps
````bash
# start android app
npm run android

# start ios app
npm run ios

# start web app
npm run web
````
